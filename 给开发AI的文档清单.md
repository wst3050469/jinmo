## 🎯 使用建议

### 首次使用Git
1. 阅读 **开发AI-Git使用指南.md** （详细教程）
2. 配置环境（Git安装、SSH密钥）
3. 克隆仓库
4. 完成一次完整的提交流程

### 日常开发
1. 参考 **开发AI-Git快速参考卡片.md** （快速查询）
2. 遵循提交信息规范
3. 定期同步最新代码

### 遇到问题
1. 查阅 **开发AI-Git使用指南.md** 的常见问题部分
2. 搜索知识库 `yongyi_system_docs`
3. 联系部署AI

## 📋 检查清单

### 首次配置
- [ ] 安装Git
- [ ] 配置Git用户信息
- [ ] 配置SSH密钥
- [ ] 添加SSH密钥到GitHub
- [ ] 测试SSH连接
- [ ] 克隆仓库

### 每次开发前
- [ ] 拉取最新代码：`git pull origin main`
- [ ] 创建功能分支：`git checkout -b feature/xxx`
- [ ] 查看当前状态：`git status`

### 每次提交前
- [ ] 检查修改的文件：`git status`
- [ ] 查看具体修改：`git diff`
- [ ] 编写清晰的提交信息
- [ ] 确认没有提交敏感文件

### 每次发布前
- [ ] 创建版本标签：`git tag -a v1.0.1 -m "说明"`
- [ ] 推送标签：`git push origin v1.0.1`
- [ ] 通知部署AI
